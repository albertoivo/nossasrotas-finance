# Nossas Rotas Finance

### Prerequisits

- Python 3
- Anaconda
- Conda
- Jupyter Notebook

### To run this project

- Clone this repo
  - `git clone https://github.com/albertoivo/nossasrotas-finance.git`
- Create the virtual environment
  - `conda env create -f environment.yml`
- Activate the environment
  - `conda activate nrf`
- Start Jupyter Notebook
  - `jupyter notebook`
